# fivem_roleplay_core
FiveM Roleplay core, Fast build FiveM serveur with this ressource.


# Features
* Car HUD
* Dead Systeme (soon dead screen)
* Player save life 
* First spawn && Player position save
* Wanted AI (true / false)
* Dirty money
* Bank money
* Wallet money
* Crosshair (true / false)
* Vocal distance systeme
* Pvp (true / false)
* Update notification
